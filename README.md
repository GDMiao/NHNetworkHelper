# NHNetworkHelper
对AFNetworking+MBProgressHUD的二次封装，使用更方便，适用性非常强：
一句话搞定网络提示；再也不用担心网络库更新后，工程要修改很多地方了！网络库更新了只需要更新这个封装好的库。
不足之处欢迎指正
